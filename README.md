# Let's Try - The Ultimate Learning Platform

A comprehensive interactive learning platform designed to help you master Computer Science concepts. Built with **React** and **Vite**, featuring a sleek user interface powered by **Tailwind CSS** and smooth animations with **Framer Motion**.

![Project Status](https://img.shields.io/badge/status-active-success.svg)
[![Live Demo](https://img.shields.io/badge/demo-online-green.svg)](https://07-ansh.github.io/Lets-Try/)

## üöÄ Features

### **1. Interactive Quizzes & Learning**
-   **Topic Selection**: Currently featuring **Operating Systems** with detailed questions.
-   **Configurable Sessions**: choose your challenge level (10, 20, 30, 50, or All questions).
-   **Instant Feedback**: Fun bubble animations ("Doing great! üéâ") and real-time score tracking.
-   **Community Quizzes**: Explore and play quizzes created by other users.
-   **Personalized Results**: Custom feedback messages based on your performance.

### **2. üí¨ Real-Time Chat System**
-   **1-on-1 Messaging**: Private, secure conversations with other users.
-   **User Search**: Find friends by Name, Username, or Email.
-   **Media Sharing**: Share **Images** and **Files** easily.
-   **Ephemeral Content**: Media files automatically disappear after **24 hours** for privacy.
-   **Unread Indicators**: Visual cues (Green Dot & Border) so you never miss a message.
-   **Global Notifications**: Receive **Toast** and **Browser** notifications even when browsing other sections.

### **3. üìù Smart Notes Library**
-   **Cloud Storage**: Securely save your study notes to Firebase Firestore.
-   **Rich Text**: Organize your thoughts with titles, tags, and content.
-   **Manage**: Edit or delete notes anytime.

### **4. üåü Experience & Design**
-   **Landing Page**: A beautiful, interactive public entry point with "Restricted View" for guests.
-   **Mobile First**: Fully responsive design with a smooth **Hamburger Menu** for mobile navigation.
-   **Profile System**: Custom Avatars and "Quiz History" tracking (Green Trophy styling).
-   **Animations**: Polished transitions using Framer Motion for a premium feel.

## üèÅ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/07-Ansh/Lets-Try.git
    cd lets-try
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Firebase Config**
    - This project uses Firebase. Ensure `src/firebase.js` is set up with your project credentials.

4.  **Start the server**
    ```bash
    npm run dev
    ```
